# Hex to RGB Converter

REST API, joka muuntaa värit Hex-muodosta RGB-muotoon.

## 1. Asennus

Ennen aloitusta, asenna tarvittavat paketit:

```bash
npm install
```

## 2. Testaus

Aja testit:

```bash
npm test
```

## 3. Käynnistys

Käynnistä palvelin:

```bash
npm start
```

## 4. Käyttö

Kun palvelin on käynnissä, voit testata sitä selaimella:
`http://localhost:3000/hex-to-rgb?hex=ffffff`
